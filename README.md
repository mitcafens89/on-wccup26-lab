# on-wccup26-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面可能涉及不同主题或用途，作为实验性、展示性或辅助性内容的集合，供参考或直接使用。

## 目录说明

```
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各个独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 共享资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：主要归档目录，每个 HTML 文件为一个独立页面，可包含其自身样式与脚本。
- **assets/**：存放多个页面共用的静态资源，便于维护和复用。

## 页面归档说明

- 每个页面均为独立 HTML 文件，可直接在浏览器中打开或部署到任意静态服务器。
- 页面之间无强依赖关系，可单独使用或修改。
- 归档内容可能随时间更新、调整或移除，恕不另行通知。
